# Lucas Portfolio Guidelines

## Reference
Visual and structural reference: guttorocha.com.br

## Goal
Create a one-page personal portfolio for Lucas Fernandes that feels as close as possible to the reference in structure, pacing, tone, interaction and visual rhythm, while using Lucas’s own identity, content and projects.

## Core Direction
This portfolio should feel:
- bold
- senior
- direct
- Brazilian
- authorial
- editorial
- modern
- dark
- digital-native
- confident
- curated

This portfolio should NOT feel like:
- a startup SaaS landing page
- a corporate agency website
- a generic portfolio template
- a Behance gallery
- a luxury serif portfolio
- a Swiss white minimalist site
- a dashboard
- a product marketing page

## Site Type
Single-page portfolio website with anchor navigation.

## Structure
The page structure must follow this sequence:
1. Header
2. Hero / Início
3. Credibility / Experience
4. Bio
5. Manifesto
6. Cases
7. Projetos próprios e experimentação
8. Fala comigo
9. Footer

## Navigation
Use a sticky top header with anchor navigation.

Menu items:
- INÍCIO
- BIO
- CASES
- FALA COMIGO

Also include a visible language switcher in the header:
- PT/BR | EN/US

The header should feel:
- thin
- subtle
- practical
- elegant
- lightly separated from the page
- not oversized
- not corporate

## Visual Style
Use a dark mode visual system.

Primary colors:
- Background primary: #0C0D0C
- Background secondary: #141514
- Border / divider / stroke: #20231F
- Primary accent / CTA / emphasis: #B9F53E

Color usage rules:
- use deep dark backgrounds
- keep strong contrast between background and text
- use muted grays for secondary text
- use the green strategically for CTA buttons, hover states, pills, highlights and key accents
- do not flood the interface with green
- cards should be dark with subtle border treatment
- overall atmosphere should feel premium, modern and slightly aggressive in a good way

## Typography
Typography should feel close in spirit to guttorocha.com.br.

Rules:
- use strong sans-serif typography
- bold hero headline
- compact uppercase labels where appropriate
- high contrast between large headings and smaller body text
- maintain editorial hierarchy
- keep line lengths readable
- avoid elegant serif luxury feeling
- avoid futuristic geometric styles
- avoid overly neutral corporate typography

## Layout and Composition
The site must feel highly editorial and not overly templated.

Rules:
- spacious hero
- strong left alignment
- confident rhythm between sections
- clear visual pauses
- not too much symmetry
- allow some tension in spacing and composition
- sections should feel intentional, not repetitive
- use subtle visual accents when needed
- preserve a sense of premium restraint

## Hero
The hero must be short, bold and memorable.

Content:
- Name: Lucas Fernandes
- Headline: Desenho produtos digitais com UX/UI para mobile e desktop.
- Supporting paragraph: Sou Product Designer com foco em UX e UI, criando produtos digitais com estrutura, consistência e clareza — de apps mobile a plataformas desktop e web.
- Metadata line: Product Designer — Brasil — UX/UI — Mobile + Desktop

Hero behavior:
- dominant typography
- minimal distractions
- strong visual presence
- no giant banner illustrations
- no overcomplicated layout
- should feel authorial and senior

- clean spacing
- dark finish

## Motion and Interaction
Motion should feel:
- smooth
- premium
- subtle
- confident
- digital-native
- not flashy

Allowed interaction patterns:
- sticky header state change on scroll
- smooth anchor scroll
- subtle reveal on scroll
- hover emphasis on case rows
- CTA hover with brightness/contrast shift
- card hover with small lift or border accent
- subtle active menu behavior

Avoid:
- exaggerated parallax
- flashy WebGL
- cartoon motion
- bouncing animations
- noisy interaction patterns

## Responsiveness
The design must remain strong on mobile and tablet.

Rules:
- preserve dark mode character
- preserve typographic impact
- stack content elegantly
- keep case list clear and legible
- keep experimentation cards strong on mobile
- make header responsive without losing identity

## Content Integrity
Always use Lucas Fernandes’s own content and projects.
Do not introduce fake projects or unrelated case studies.

Projects currently used:
- Auralize
- SmartDyn
- Arvorar
- AsaVac
- LoveMyPet

## Final Rule
Whenever there is ambiguity, prioritize:
1. fidelity to guttorocha.com.br structure and rhythm
2. Lucas Fernandes’s content
3. dark editorial authorial tone
4. clarity and strong hierarchy
5. premium modern execution
