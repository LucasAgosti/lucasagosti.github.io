Create a premium portfolio Projects page for Lucas Fernandes, a Product Designer focused on UX/UI for mobile and desktop digital products.

The design should match the homepage visual language and feel part of the same portfolio system. It should be inspired by the simplicity, sophistication and curation style of serenacrq.com, but adapted to Lucas’s own portfolio identity and project content.

GOAL
Design a curated projects hub page that presents Lucas’s selected work in a polished, premium and highly intentional way. The page should feel editorial, minimal, visually refined and easy to scan, while making each project feel valuable and well-crafted.

STYLE DIRECTION
- Premium minimalist
- Editorial layout
- Spacious
- Sophisticated and calm
- Modern personal portfolio
- Strong hierarchy
- Large margins
- Soft luxury aesthetic
- Light background with refined typography
- Gentle separators and subtle visual structure

DO NOT
- Do not create a generic portfolio grid with identical cards only
- Do not make it feel like a Behance gallery
- Do not create a startup case-study directory
- Do not add unnecessary filters unless they are extremely elegant
- Do not overcomplicate interactions
- Do not make it dense or crowded

PAGE STRUCTURE

1. HEADER
Use the same header system as the homepage:
- Lucas Fernandes
- Nav: Home, Projects, About
- Right-side CTA: Resume or Let’s Connect

Keep full consistency with the homepage style.

2. PAGE INTRO / HERO
Create a clean intro section for the page.

Include:
- Eyebrow or small label: Selected Work
- Main title: Selected Projects
- Supporting paragraph:
  “A curated collection of product design, UX and UI work across mobile and desktop experiences. These projects reflect my approach to structure, usability, systems and interface quality.”

This intro should feel elegant and concise.
Do not over-design it.
Let typography and spacing do the work.

- Make the projects feel premium and visually curated
- Use a strong image/mockup area for each project
- Include project title, category, platform and description
- Add a subtle call-to-action like View Case

- Allow some projects to feel more prominent than others, especially:
  Auralize, SmartDyn and Arvorar
- Keep visual rhythm dynamic but cohesive
- Avoid repetitive identical-card fatigue
- Use spacing and composition to imply project importance
- The page should feel like a thoughtful portfolio exhibition
